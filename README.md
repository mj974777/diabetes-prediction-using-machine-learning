# diabetes-prediction-using-machine-learning 
Mohammed Ahmed Jibril (202409010800)


Overview
This project focuses on predicting the likelihood of diabetes in patients using Machine Learning techniques. The goal is to build an intelligent system that analyzes medical data to provide early diagnostic predictions.

 Data and Preprocessing
 Data cleaning and preprocessing were performed to ensure high-quality training.
 Applied Feature Scaling techniques to optimize algorithm performance.
 Conducted Exploratory Data Analysis (EDA) to identify key factors influencing diabetes.

 Models and Results
 Algorithm Used: Logistic Regression (stored in the .pkl file).
 *Model Accuracy:* The project achieved an accuracy of 76%.
  Evaluation was conducted using metrics such as the Confusion Matrix to ensure prediction reliability.

 Project Structure
 preprocessing.ipynb: The comprehensive notebook for data analysis and model training.
 app.py: The source file for the interactive user interface.
 best_diabetes_model_logistic_regression.pkl: The saved file of the final trained model.
 Reports and Presentation: Detailed project documentation and slides.

 Technologies Used
Python (Pandas, NumPy, Scikit-learn).
 Streamlit (for the user interface).
 GitHub (for version control and hosting).
